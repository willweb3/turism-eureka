## Notas Importantes

1. **Integration API vs Marketplace API:**
   - Integration API: Server-side, usado para buscar/atualizar dados de usuários existentes
   - Marketplace API: Client-side, usado para autenticação e registro de novos usuários

2. **Segurança:**
   - Nunca exponha `SHARETRIBE_CLIENT_SECRET` no client-side
   - Use `NEXT_PUBLIC_*` apenas para dados públicos seguros
   - As credenciais de Integration API não têm permissão para criar usuários
